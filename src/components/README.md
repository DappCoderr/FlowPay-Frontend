# Atomic Design Structure

- **atoms** — Basic UI elements: `Button`, `Logo`, `Badge`
- **molecules** — Composed atoms: `IconLabel`, `Modal`, `Card`, `StepItem`
- **organisms** — Sections: `Header`, `Footer`, `HowItWorksModal`, `HeroSection`
- **templates** — Layouts: `MainLayout`, `LandingTemplate`

Use `@/components` or `@/components/atoms/Button` etc. for imports.
